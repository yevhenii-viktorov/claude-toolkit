# business-research

A 7-prompt playbook for founder-grade business research. Each prompt is honed to produce specific, defensible, decision-grade output instead of generic LLM mush.

## When to use

- Researching a market
- Stress-testing a business idea
- Building financial projections
- Writing an executive summary or pitch deck
- Competitor analysis
- Pressure-testing a go-to-market plan

The skill can run the full sequence or any single piece in isolation.

## The 7 prompts

1. Market research
2. Model critique
3. Financials
4. Executive summary
5. Competitor research
6. Pitch deck
7. GTM stress test

Each lives in [`references/`](./references) as a standalone prompt.

## Install

```bash
cp -r business-research ~/.claude/skills/
```

Then invoke with `/business-research` in Claude Code.
